# Monster Strike Auto 
The scripts are written with autojs.

## AutoJs Version
4.1

## Usage
1. Download Auto.js.4.1.apk to your mobile phone.
2. Install it.
3. Save the scripts to your autojs App.
4. Run your game.
5. Run the script.
6. Enjoy! 

## Note
Some arguments need to be modified to fit your phone.
